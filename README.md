# ML4T 2018

This repository contains my solutions to the course [Machine Learning for Trading](http://quantsoftware.gatech.edu/CS7646_Fall_2018) taught by professor [Tucker Balch](https://www.linkedin.com/in/tuckerbalch/) at [Georgia Tech](https://www.gatech.edu/) that I attended in the Fall of 2018.



Every assignment has its own readme file with an explanation of the assignment.

* [Martingale](http://quantsoftware.gatech.edu/martingale) (difficulty level: easy)
* [Optimize_something](http://quantsoftware.gatech.edu/optimize_something) (difficulty level: easy)
* [Assess_learners](http://quantsoftware.gatech.edu/assess_learners) (difficulty level: challenging)
* [Defeat_learners](http://quantsoftware.gatech.edu/defeat_learners) (difficulty level: easy)
* [Marketsim](http://quantsoftware.gatech.edu/marketsim)  (difficulty level: moderate)
* [Manual_strategy](http://quantsoftware.gatech.edu/manual_strategy) (difficulty level: moderate)
* [Qlearning_robot](http://quantsoftware.gatech.edu/qlearning_robot)  (difficulty level: moderate)
* [Strategy_learner](http://quantsoftware.gatech.edu/strategy_learner) (difficulty level: very challenging)
